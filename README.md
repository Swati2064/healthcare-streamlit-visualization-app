# healthcare-streamlit-visualization-app
# 🏥 Healthcare Data Visualization Dashboard

An interactive **Streamlit web application** for exploring and visualizing healthcare (heart disease) data using **Seaborn and Matplotlib**.

This project helps users understand patterns in heart disease data through interactive charts and filters, making data analysis simple and visual.

---

## 📌 Features

- 📊 Interactive data visualization dashboard
- 🎛️ Sidebar filter for Age range selection
- ❤️ Chest Pain Type vs Heart Disease analysis
- 📈 Age distribution visualization
- 🧪 Cholesterol level comparison with target variable
- ⚡ Real-time updates based on user input
- 🎨 Clean and simple UI using Streamlit

---

## 🛠️ Technologies Used

- Python 🐍
- Streamlit 🌐
- Pandas 📊
- Seaborn 🎨
- Matplotlib 📉
